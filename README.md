# Real-Time-Big-Data-Pipeline
## 📌 Overview
An end-to-end real-time big data pipeline that ingests user data from the Random User API, processes streaming data, stores it in distributed systems, and visualizes results in real time.

---

## 📖 Project Description
This project simulates a production-grade streaming environment using modern Big Data technologies. Data is continuously collected from the Random User API, streamed through Apache Kafka, processed using Apache Spark, and stored in multiple systems for analytics and visualization.

The pipeline supports:
- Real-time data ingestion
- Distributed stream processing
- Scalable data storage
- Real-time monitoring and visualization
- Batch and streaming analytics
