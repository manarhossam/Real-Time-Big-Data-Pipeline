# Real-Time-Big-Data-Pipeline
📌 Overview
End-to-end real-time data pipeline that ingests user data from Random User API, processes streaming data, and visualizes results in real time.

📌 Description
This project is a real-time big data pipeline that simulates a production streaming environment. It ingests data from the Random User API, processes it using distributed systems, and stores results for real-time analytics and visualization.

🖥️ Deployment

Deployed and tested on a Linux Virtual Machine (VM) in a real-time streaming environment
